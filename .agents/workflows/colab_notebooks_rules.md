# Rules for Colab Notebooks

1. **Do not create multipe notebooks**: Avoid creating numerous new notebooks in Google Colab unnecessarily.
2. **Reuse Existing Notebooks**: Update or use the ones you have already created instead of starting fresh.
3. **Naming Convention**: Always give meaningful, descriptive names to your Colab notebooks to easily identify their purpose.
4. **Log Notebook Info**: Save the names, links, and any running details of the Colab notebooks in logs (e.g., inside `logs/` directory or appropriate logging files) so they are kept track of.
